# An Educational Java Virtual Machine

## 使用方法
准备工作
```
# 解压rt.jar到指定文件夹
cd ${dir-rt-extract} && jar -xvf rt.jar
```
```
# Clone a repo
$ git clone https://github.com/chaomaer/cpp-jvm
$ cd cpp-jvm

# Build cpp-jvm
$ mkdir build
$ cmake .. （如果是Mac系统，需要加-DAPPLE=1）
$ make

# 运行class文件
$ cd build/
$ ./cpp-jvm {dir-rt-extract} {path-of-class} ${name-of-class}
```
